# Babatunde Oyedele - Professional Portfolio

Welcome to the repository for my professional portfolio website! This site serves as a digital resume and a showcase of my skills, experience, and services as an Information Systems Control Specialist and Cyber Security Assurance Professional.

## 🌐 Live Site
You can view the live website here: [https://tundeoyedele.netlify.app/](https://tundeoyedele.netlify.app/)

## 👤 About Me
I am a results-driven Information Technology and Cyber Security Assurance Officer with years of hands-on experience in conducting comprehensive IT assurance assessments. My expertise lies in evaluating and enhancing the security posture of enterprise IT environments, including logical access management, privileged access management, application and database security, and network infrastructure.

## 🚀 Key Features
- **Dynamic Typing Header**: Engaging introduction using Typed.js.
- **Skills Section**: Visual representation of my technical skills and proficiencies.
- **Experience Timeline**: Detailed history of my professional roles and achievements at organizations like Keystone Bank and Stanbic IBTC Bank.
- **Services Offered**: Information on how I can help businesses with IT Risk Analysis, System Assurance, IT Auditing, and Script Writing/Training.
- **Portfolio Gallery**: A filterable gallery of my past projects and areas of expertise.
- **Awards Carousel**: Highlights of my professional recognitions and awards.
- **Working Contact Form**: A functional contact form powered by Fetch API and Google Forms for seamless communication.

## 🛠️ Technologies Used
This website was built using modern web technologies to ensure a responsive and interactive user experience:
- **HTML5 & CSS3**
- **Bootstrap 5** (for responsive layout and styling)
- **JavaScript & jQuery** (for interactivity and DOM manipulation)
- **Google Fonts** & **FontAwesome/Bootstrap Icons** (for typography and iconography)
- **Libraries/Plugins**:
  - `Typed.js` (for typing animation effect)
  - `OwlCarousel` (for the testimonials/awards slider)
  - `Isotope` (for portfolio filtering)
  - `Lightbox` (for image viewing)
  - `WOW.js` & `Animate.css` (for scroll animations)
  - `Waypoints` & `CounterUp` (for animated stats)

## ✉️ Contact Integration
The contact form on this site is custom-built to submit data directly to a Google Sheets spreadsheet using Google Forms as the backend. It uses the `fetch` API and `FormData` with `mode: 'no-cors'` to ensure reliable and secure message delivery without the need for a traditional backend server.
